# 📊 Retail Sales Dashboard (Excel)

## 🔹 Project Overview

This project is an interactive sales dashboard built in Microsoft Excel. It analyzes sales performance using pivot tables, slicers, and charts.

## 🔹 Features

* Data Cleaning using Power Query
* Sales, Cost, and Profit Calculation
* Region-wise Analysis
* Interactive Dashboard with Slicers

## 🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* Power Query
